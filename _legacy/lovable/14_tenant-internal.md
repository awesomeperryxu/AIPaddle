在当前项目中，请生成"租户内部管理"页面。这是企业租户管理员管理自己组织内部的页面，包含多个子模块。

## 页面信息
- 页面标题：内部管理
- 副标题：管理企业组织架构、人员和权限
- 布局：左侧子导航（w-48, border-r）+ 右侧内容区（flex-1 p-5）

## 左侧子导航

垂直导航列表（space-y-1），每项（px-3 py-2 rounded-md text-sm cursor-pointer）：
- 组织架构（Building2 图标）
- 人员管理（Users 图标）
- 角色权限（Shield 图标）
- 访问控制（Lock 图标）
- 资源配额（BarChart3 图标）
- 审计日志（FileText 图标）

选中态：bg-primary/10 text-primary font-medium
悬浮态：bg-muted

---

## 子页面 1：组织架构

### 布局：左右分栏（flex gap-6）

**左侧部门树（w-72）**：
- 顶部：标题"部门结构" + 添加部门按钮(Plus, size="sm")
- Tree 组件（递归渲染）：
  - 每个节点：展开/折叠箭头(ChevronRight) + 文件夹图标(Folder) + 部门名 + 人数Badge
  - 选中节点：bg-primary/10 rounded-md
  - 缩进：每级 pl-4
  - 右键菜单：编辑 / 添加子部门 / 删除

示例树结构：
```
示范科技有限公司 (1245)
├── 技术部 (456)
│   ├── 前端组 (120)
│   ├── 后端组 (180)
│   └── AI 组 (156)
├── 产品部 (89)
├── 运营部 (234)
├── 人力资源部 (67)
└── 财务部 (45)
```

**右侧部门详情（flex-1）**：
- 部门信息 Card：部门名称 / 部门编码 / 负责人 / 上级部门 / 创建时间
- 资源配额 Card：Token 月限额(Progress) / 存储限额(Progress) / Agent 数量限额
- 部门成员列表（简表，5 行）：头像+姓名 / 职位 / 角色 / 状态

## 子页面 2：人员管理

### 布局：垂直堆叠

**操作栏**：
- 左侧：部门筛选 Select（树形下拉）+ 状态筛选 Select（全部/正常/试用/离职/冻结）+ 搜索框
- 右侧：添加用户按钮 + 批量导入按钮(Upload) + 企微同步按钮(RefreshCw)

**用户列表 Table**：
- 列：头像+姓名 / 工号 / 部门 / 职位 / 角色(多Badge) / 状态 / 最后登录 / 操作
- 状态：正常(绿) / 试用(蓝) / 离职(灰) / 冻结(红)
- 操作 DropdownMenu：编辑 / 调岗 / 重置密码 / 冻结 / 离职处理(text-destructive)

**添加用户 Dialog（520px）**：
- 工号 / 姓名 / 邮箱 / 手机号
- 所属部门（树形 Select）/ 直属上级（搜索 Select）/ 职位
- 角色分配（多选 Checkbox）
- 是否发送激活邮件（Switch）

## 子页面 3：角色权限

### 布局：左右分栏

**左侧角色列表（w-64）**：
- 标题"角色列表" + 创建角色按钮
- 预设角色（带 Lock 图标，不可删除）：
  - 企业超级管理员
  - 企业管理员
  - AI 管理员
  - 部门管理员
  - 开发者
  - 普通用户
- 自定义角色（可编辑/删除）
- 选中角色高亮

**右侧权限矩阵（flex-1）**：
- 顶部：角色名 + 描述 + 编辑按钮
- 权限表格（Table）：
  - 行 = 功能模块（Agent管理/Skill管理/知识库/工作流/成员管理/安全审核）

  - 单元格 = Checkbox
- 数据权限区域：RadioGroup（全部数据 / 本部门及下级 / 本部门 / 仅本人）

## 子页面 4：访问控制

### 布局：Tab 切换（4 个 Tab）

**Tab 1 - SSO 配置**：
- 已配置的 IdP 卡片列表（grid grid-cols-3 gap-4）：
  - 每张：Logo + 名称 + 状态Badge(已启用=绿/未配置=灰) + 配置按钮
  - IdP 列表：Azure AD / Okta / Google Workspace / 企微 / 飞书 / 钉钉

**Tab 2 - MFA 策略**：
- 全局 MFA 开关（Switch）
- MFA 方式（Checkbox 组）：短信验证码 / 邮箱验证码 / TOTP 应用
- 策略级别 RadioGroup：企业级 / 角色级 / 用户级

**Tab 3 - 会话管理**：
- 会话超时（Slider：30分钟-24小时）
- 最大并发会话数（Input type="number"）
- 在线会话列表 Table：用户 / 设备 / IP / 登录时间 / 操作(强制登出按钮)

**Tab 4 - 登录安全**：
- 密码策略 Card：最小长度(Slider) / 复杂度要求(Checkbox组) / 有效期(Select)
- 登录失败锁定 Card：失败次数阈值(Input) / 锁定时长(Select)
- IP 白名单 Card：IP 列表 + 添加 Input + 删除按钮

## 子页面 5：资源配额

### 布局：垂直堆叠

**顶部统计（grid grid-cols-3 gap-4）**：
- 企业总配额 Card：Token 总量 + 存储总量 + Agent 总数
- 已分配 Card：已分配给部门的量
- 未分配 Card：剩余可分配量

**配额分配 Table**：
- 列：部门名称 / Token 月限额 / Token 已用(Progress) / 存储限额 / 存储已用(Progress) / Agent 数限额 / 操作
- 操作：调整配额按钮 → Dialog（当前配额 + 新配额 Input + 调整原因 Textarea）

**配额监控图表（grid grid-cols-2 gap-4）**：
- 左侧：使用趋势 LineChart（近 30 天）
- 右侧：部门用量排行 BarChart（水平）

## 子页面 6：审计日志

### 布局：垂直堆叠

**筛选栏（flex gap-3 flex-wrap）**：

- 操作类型颜色：登录=蓝 / 用户管理=紫 / 角色变更=橙 / 配额调整=青 / 资源访问=绿
