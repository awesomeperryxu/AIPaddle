在工作流编辑器中，请添加以下交互功能。

## 1. 右键菜单

### 画布空白区域右键
ContextMenu 组件，菜单项：
- 添加节点（打开左侧节点面板）
- 粘贴节点
- 分隔线
- 运行工作流
- 分隔线
- 添加便签
- 添加评论
- 分隔线
- 导出 DSL
- 导入 DSL

### 节点右键
- 运行此步骤（Play 图标）
- 更换节点类型 → 子菜单
- 分隔线
- 复制（⌘C）
- 复制副本（⌘D）
- 删除（Delete）

### 多选节点右键
- 对齐：上/中/下/左/居中/右
- 分布：水平均分/垂直均分

## 2. 评论系统

### 评论模式按钮（底部工具栏）
- 点击后光标变为评论图标
- 点击画布放置评论标记

### 评论标记（画布上）
- 小圆形图标（w-6 h-6, bg-primary, 白色 MessageSquare 图标）
- 悬浮显示评论预览（Popover，最多 2 行）
- 点击展开评论线程

### 评论线程（Popover，w-80）
- 顶部：评论者头像 + 姓名 + 时间 + 解决按钮(CheckCircle) + 关闭
- 评论内容
- 回复列表
- 底部：回复 Input + 发送按钮

## 3. 便签节点

### 视觉
- 矩形区域（默认 200x150），无连接点
- 半透明彩色背景（6 种：blue/cyan/green/yellow/pink/violet）
- 可拖拽调整大小（右下角手柄）

### 编辑
- 双击进入编辑模式
- 顶部工具栏：字体大小 Select + 加粗/斜体 + 颜色切换
- 点击外部退出编辑

## 4. 版本历史面板（右侧，w-80）

### 触发
点击顶部工具栏的 History 按钮

### 布局
- 顶部：标题"版本历史" + 关闭按钮
- 版本列表：每项（border-b py-3）
  - 版本号 + 时间 + 操作人
  - 状态 Badge（Published=绿/Draft=灰）
  - 操作：恢复按钮 + 删除按钮
- 恢复确认 Dialog

## 5. 环境变量面板（右侧，w-80）

### 触发
点击顶部工具栏的 Variable 按钮

### 布局
- 顶部：标题"环境变量" + 添加按钮 + 关闭
- 变量列表：每行（名称 + 类型Badge(string=蓝/number=绿/secret=红) + 值 + 编辑/删除）
- secret 类型值显示 ****
- 添加/编辑 Dialog：名称 + 类型 Select + 值 Input + 描述

## 6. 检查清单（Popover，w-80）

### 触发
点击顶部工具栏的 CheckCircle 按钮（带数字角标）

### 布局
- 标题"检查清单" + 问题总数
- 问题列表：每项（图标 + 节点名 + 问题描述）
  - 红色图标：插件缺失
  - 黄色图标：配置问题
- 点击跳转到问题节点
- 全部通过：绿色勾 + "所有检查通过"

## 7. DSL 导入/导出

### 导出 Dialog
- 标题"导出工作流"
- Checkbox：是否包含密钥值
- 导出按钮（下载 .yml 文件）

### 导入 Dialog
- 标题"导入工作流"
- 文件上传区域（拖拽或点击）
- 验证状态
- 确认导入按钮
