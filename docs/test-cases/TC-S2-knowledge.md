## 金标准问答集（golden set）建设规范

- 位置：`tests/fixtures/golden-set.json`；首版 ≥20 题，随语料扩充
- 每题字段：`question`（问题）、`expected_points`（必须命中的要点，1-3 条）、`expected_sources`（应引用的文档）、`type`（事实/汇总/拒答）
- 题型配比：事实型 60% / 跨段汇总型 25% / 应拒答型 15%
- 评分：要点命中（LLM 辅助判定 + 关键词兜底）+ 引用核对（程序断言）
- **回归纪律**：任何改动检索逻辑、提示词、切块策略的 Issue（`flag:rag`）都必须重跑全集，通过率不得低于上一版（防退化）
