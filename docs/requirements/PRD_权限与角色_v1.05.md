## 2.11.1 设计原则

1. **两层体系**：平台侧（AIPaddle 运营方）与租户侧（企业内）角色完全隔离；平台角色不可见租户业务数据明细（仅用量/账单等运营聚合数据）。
2. **RBAC + 数据范围正交**：功能权限（页面 × 查看/创建/编辑/删除/审批）与数据范围（全部 / 本部门及下级 / 本部门 / 本人）独立配置。
3. **最小权限默认**：新用户默认「普通员工」；提权需审批并记录审计。
4. **全程审计 + MFA**：角色/权限变更、导出、删除均审计留痕，不可篡改；管理员敏感操作强制二次验证（TOTP）。
5. **双人复核（四眼原则，P5 确认）**：仅「管理员类角色」的授予/回收需第二名管理员复核生效；其余角色变更单人操作 + 审计。

## 2.11.2 数据范围定义

| 范围 | 含义 | 典型角色 |

| 全部数据 | 本租户全量 | 企业超管、企业管理员、AI/安全/财务管理员 |

## 2.11.5 资源创建与共享权限（本次确认核心新增）

### 创建权限总表

| 资源类型 | 可创建人 | 默认可见范围 | 共享/上架路径 |
|---|---|---|---|
| 知识库 | AI 管理员（企业级）；开发者（部门级） | 创建者所在范围 | 绑定 Agent/WF 时受穿透权限约束（2.11.6） |
| Skill L1 企业通用 | 系统管理员（AI 管理员及以上） | 全员**强制安装** | 直接下发，Skill Hub「强制安装」区 |
| Skill L2 个人 | 每位员工 | 仅本人（我的 Skill） | 不共享 |
| Skill L3 上架 | 每位员工发起 | 审核通过后全员可安装 | 个人提交 → 系统管理员审核 → 上架 Skill Hub |
| Workflow / Chatflow | 每位员工（个人草稿） | 以**部门为单位**共享 | 创建 → 系统管理员审核 → 共享至本部门或全企业 |
| Agent | 每位员工（个人草稿）；部门管理员创建需 AI 管理员审批（P2 确认） | 以**部门为单位**共享 | 同 Workflow：审核后共享至部门/全企业 |
| 企业级 Multi-Agent 编排（跨部门调用多 Agent 的 WF/CF） | **仅系统管理员** | 全企业 | 系统管理员创建并持续管理 |
| MCP | 租户管理员申请注册 | 审批后按 scope 生效 | 租户提交 → 平台安全审计审批（模式 B，见 2.9.8） |

### Skill 三级体系（确认稿）

- **L1 企业通用**：系统管理员创建，强制安装、全员可用、用户不可卸载（如敏感信息检测）。
- **L2 个人**：员工自建自用，不进入市场，不可被他人调用。
- **L3 市场上架**：员工将 L2 提交上架申请 → 系统管理员审核（含安全扫描：敏感数据/非法指令/DB 风险）→ 上架 Skill Hub 供全员安装；驳回需注明原因。

### Workflow/Chatflow 与 Agent 的部门共享模型

员工创建的 WF/CF/Agent 默认为个人草稿（仅本人可调试）；提交共享申请后由系统管理员审核，可选共享范围：**本部门 / 全企业**。共享后的资源由创建人维护、系统管理员可强制下线；跨部门企业级 Multi-Agent 编排一律由系统管理员创建与管理。

## 2.11.6 知识库穿透权限（引用链访问控制）

1. **默认模式 = 调用者权限继承**：Agent/WF/Skill 运行时，以**最终用户身份**校验其绑定知识库的 ACL；用户无权限的知识库不参与召回（静默降级 + 记录审计），杜绝「借 Agent 越权读库」。
2. **可选模式 = 资源自带授权**：资源 Owner 可显式声明「知识库随资源共享」（用于客服 FAQ 等公共库），需 AI 管理员 + 知识库 Owner 双确认，并在资源详情标注、全程审计。
3. **交集校验**：引用链每一跳按（用户权限 ∩ 资源授权 ∩ 知识库 ACL）取最小集，多层嵌套（Agent→WF→Skill→KB）不得绕过；导出/下载永远按用户本人权限执行。

## 2.11.7 审批与复核流汇总

| 场景 | 提交人 | 审批人 | 生效 |
|---|---|---|---|
| Skill L3 上架 | 员工 | 系统管理员（含安全扫描） | 上架 Skill Hub |
| WF/CF/Agent 共享 | 创建人 | 系统管理员 | 部门或全企业可见 |
| 部门管理员创建 Agent（P2） | 部门管理员 | AI 管理员 | 创建成功 |
| 管理员类角色授予/回收（P5） | 管理员 A | 管理员 B 复核 | 双签后生效 |
| MCP 注册（Production） | 租户管理员 | 平台安全审计 | 按 scope/限流生效 |
| 部门配额调整 | 部门管理员 | 企业管理员 | 立即生效 + 审计 |
