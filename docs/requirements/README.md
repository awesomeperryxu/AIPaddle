# AIPaddle 需求文档

## 当前基准

- `PRD_Core_v1.04.md`：完整产品需求基准
- `UI_Design_Spec_v1.04.md`：前端技术与 UI 设计基准
- `modules/`：独立模块需求
- `iteration/`：迭代需求原始记录
- `ui-pages/`：复杂页面的详细交互方案

旧版重复 PRD、旧版 UI 规范和特定生成平台的提示词已删除。功能完成度以 `../IMPLEMENTATION_AUDIT.md` 为准；PRD 中的 MVP 完成状态仅代表前端原型覆盖，不代表后端业务闭环完成。

问题与变更通过 [AIPaddle GitHub Issues](https://github.com/HELLOPERRYXU/AIPaddle/issues) 管理。
