## 🎯 核心功能模块

### 工作台
- **监控面板** - 实时统计和趋势分析
- **个人助理** - AI 助手对话界面

### Agent 管理
- **Agent 管理** - 创建、配置、发布 AI Agent
- **数字员工** - 与已发布 Agent 交互

### Skill 生态
- **Skill Hub** - 企业 AI 能力市场
- **我的 Skill** - 个人创建的 Skill

### 知识库
- **知识库管理** - 文档管理和索引
- **知识库问答** - 基于知识库的问答测试

### 工作流
- **工作流管理** - 可视化工作流编辑器
- 支持 12 种节点类型

### 安全与管理
- **安全管理** - 审核和风险评估
- **成员管理** - 用户和权限管理

### 平台管理
- **租户管理** - 多租户 SaaS 管理
- **运营看板** - 平台运营数据

## 🛠️ 技术栈

### 前端
- **框架**: Next.js 16.2.4
- **语言**: TypeScript 5.7.3
- **样式**: Tailwind CSS 4.2.0
- **UI 组件**: Radix UI + shadcn/ui
- **图标**: Lucide React
- **图表**: Recharts 2.15.0
- **表单**: React Hook Form + Zod

### 后端（待开发）
- 用户认证系统
- RESTful API
- 工作流引擎
- 知识库向量化
- 权限管理系统
