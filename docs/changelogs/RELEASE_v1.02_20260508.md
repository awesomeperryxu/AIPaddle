**核心功能**:

4. 访问控制（SSO、MFA、会话管理、登录策略）
5. 资源配额管理（三级配额、监控告警）
6. 审计日志（操作审计、查询导出、保留策略）

**技术亮点**:
- RBAC + ABAC 混合权限模型
- 4 种数据权限级别（全部/本部门+/本部门/本人）
- 支持 6 种 IdP（Azure AD、Okta、Google、企微、飞书、钉钉）
- 3 种 MFA 方式（短信、邮箱、TOTP）
- 时序数据库存储审计日志
